
Q:   What the program does?
Ans: Program to find the maximum difference of j-i such that a[j]>a[i],
     for a given array of 'n' elements.

Q:   What is sample input and corresponding output?
Ans: 
     Enter the size of array
     7
     Enter array elements
     2  4  8  1  3  7  2
     Array is:
     2 4 8 1 3 7 2 
     Maximum difference is 6.

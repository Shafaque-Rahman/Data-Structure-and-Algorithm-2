
Q:   What the program does?
Ans: This program will find the largest sub-array of given array having
     equal number of 0's and 1's.

Q:   What is sample input and output?
Ans: 
                             OUTPUT
     Enter size of array
     6
     Enter array elements
     1  1  0  1  0  1
     The largest sub-array is:-
     1  0  1  0
